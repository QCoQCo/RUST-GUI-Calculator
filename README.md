## 한국어

### 개요
Rust와 Iced 프레임워크를 사용하여 개발된 데스크톱 GUI 계산기 애플리케이션입니다.

![Calculator Screenshot](demo/스크린샷%202026-02-09%20오후%207.46.17.png)

### 기능
- **기본 사칙연산**: 덧셈(+), 뺄셈(-), 곱셈(×), 나눗셈(÷)
- **소수점 계산**: 소수점 이하 숫자 계산 지원
- **부호 변경**: ± 버튼으로 양수/음수 전환
- **퍼센트 계산**: % 버튼으로 퍼센트 계산
- **에러 처리**: 0으로 나누기 등 잘못된 연산 시 에러 표시
- **긴 숫자 처리**: 긴 숫자 입력 시 자동 폰트 크기 조정

### 기술 스택
- **언어**: Rust
- **GUI 프레임워크**: Iced 0.10
- **에디션**: Rust 2021

### 빌드 및 실행

#### 사전 요구사항
- Rust (최신 안정 버전)
- Cargo (Rust 패키지 매니저)

#### 실행 방법
```bash
# 프로젝트 디렉토리로 이동
cd gui_calculator

# 실행
cargo run
```

#### 릴리즈 빌드
```bash
# 최적화된 릴리즈 빌드
cargo build --release

# 실행 파일 위치: target/release/gui_calculator
```

### 사용 방법
1. 숫자 버튼을 클릭하여 숫자 입력
2. 연산자 버튼(+, -, ×, ÷)을 클릭하여 연산 선택
3. = 버튼을 클릭하여 결과 확인
4. C 버튼으로 초기화
5. ± 버튼으로 부호 변경
6. % 버튼으로 퍼센트 계산

### 프로젝트 구조
```
gui_calculator/
├── src/
│   └── main.rs          # 메인 애플리케이션 코드
├── demo/                # 데모 스크린샷
├── Cargo.toml          # 프로젝트 설정 및 의존성
└── README.md           # 프로젝트 문서
```

### 특징
- **다크 테마**: 눈에 편안한 다크 테마 UI
- **반응형 디자인**: 버튼 호버 및 클릭 효과
- **직관적인 UI**: 색상으로 구분된 버튼 타입
  - 숫자 버튼: 어두운 회색
  - 연산자 버튼: 주황색
  - 기능 버튼: 밝은 회색
  - 등호 버튼: 파란색

## 日本語

### 概要
RustとIcedフレームワークを使用して開発されたデスクトップGUI電卓アプリケーションです。

![Calculator Screenshot](demo/스크린샷%202026-02-09%20오후%207.46.17.png)

### 機能
- **基本四則演算**: 加算(+)、減算(-)、乗算(×)、除算(÷)
- **小数点計算**: 小数点以下の数値計算をサポート
- **符号変更**: ±ボタンで正数/負数の切り替え
- **パーセント計算**: %ボタンでパーセント計算
- **エラー処理**: ゼロ除算などの不正な演算時にエラー表示
- **長い数字の処理**: 長い数字入力時に自動フォントサイズ調整

### 技術スタック
- **言語**: Rust
- **GUIフレームワーク**: Iced 0.10
- **エディション**: Rust 2021

### ビルドと実行

#### 前提条件
- Rust (最新の安定版)
- Cargo (Rustパッケージマネージャー)

#### 実行方法
```bash
# プロジェクトディレクトリに移動
cd gui_calculator

# 実行
cargo run
```

#### リリースビルド
```bash
# 最適化されたリリースビルド
cargo build --release

# 実行ファイルの場所: target/release/gui_calculator
```

### 使用方法
1. 数字ボタンをクリックして数字を入力
2. 演算子ボタン(+, -, ×, ÷)をクリックして演算を選択
3. =ボタンをクリックして結果を確認
4. Cボタンでリセット
5. ±ボタンで符号を変更
6. %ボタンでパーセント計算

### プロジェクト構造
```
gui_calculator/
├── src/
│   └── main.rs          # メインアプリケーションコード
├── demo/                # デモスクリーンショット
├── Cargo.toml          # プロジェクト設定と依存関係
└── README.md           # プロジェクトドキュメント
```

### 特徴
- **ダークテーマ**: 目に優しいダークテーマUI
- **レスポンシブデザイン**: ボタンのホバーとクリック効果
- **直感的なUI**: 色で区別されたボタンタイプ
  - 数字ボタン: 濃い灰色
  - 演算子ボタン: オレンジ色
  - 機能ボタン: 明るい灰色
  - 等号ボタン: 青色
